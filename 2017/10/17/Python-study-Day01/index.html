      1 Helloworld小程序

      欢迎来到python的世界，首先向程序世界打招呼：

      print ('Hello World!')

      我们可以使用 sublime Text 编辑一个文本文件保存为HelloWorld.py ,再在同目录下使用 python HelloWorld.py 运行这个脚本。

      # !/usr/bin/env python
# HelloWorld.py

print ("Hello World!")

      2 变量

      name="whm"
name2=name
print("My name is ",name,name2)

name="Paoche Ge"
print("My name is ",name,name2)

      输出结果：

      My name is Whm Whm

      My name is PaoChe Ge Whm

      输出结果表明：name2不是指向name，而是指向”whm”,即使name指向内容改变，name2也不会改变

      11 dict

      定义一个dict，字典数据类型，查找快速，但是需要占用的空间较大（典型的以空间换取时间）

      >>> d = {'Michael': 95, 'Bob': 75, 'Tracy': 85}
>>> d['Michael']
95

      判断dict变量d中是否存在”Thomas”-key

      >>> 'Thomas' in d
False


>>> d.get('Thomas')      #默认不存在返回None
>>> d.get('Thomas', -1)  #指定不存在key默认返回值为-1
-1

      删除”Bob” 键值对

      >>> d.pop('Bob')
75
>>> d
{'Michael': 95, 'Tracy': 85}

      dict的key键必须是不可变的，如：字符串，整数

      12 set

      #set需要传入一个list作为输入集合    
>>> s = set([1, 2, 3])
>>> s
{1, 2, 3}

      set可添加元素，但是元素重复便不会继续添加，而且set属于 无序 集合

      >>> s.add(4)
>>> s
{1, 2, 3, 4}
>>> s.add(4)
>>> s
{1, 2, 3, 4}

      set可以删除元素

      >>> s.remove(4)
>>> s
{1, 2, 3}

      set可以做数学集合运算

      >>> s1 = set([1, 2, 3])
>>> s2 = set([2, 3, 4])

>>> s1 & s2
{2, 3}
>>> s1 | s2
{1, 2, 3, 4}

      13 不可变对象再学

      >>> a = 'abc'
>>> b = a.replace('a', 'A')
>>> b
'Abc'
>>> a
'abc'

      a.replace(“”,””)只是创建了一个新的字符串，并没有改变a的指向，所以会发现a和b的值不同。
